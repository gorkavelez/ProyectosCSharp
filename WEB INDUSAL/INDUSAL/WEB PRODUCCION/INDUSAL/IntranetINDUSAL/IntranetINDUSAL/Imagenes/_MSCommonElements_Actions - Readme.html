Biblioteca de imágenes de Visual Studio: elementos comunes (acciones)

24bit Art
Esta imagen es en color de 24 bits. El formato del archivo .png incluye una capa transparente para permitir cortar y pegar en imágenes nuevas con facilidad, e incluye varios tamaños para cada elemento. La forma de ver una vista previa de estas imágenes compuestas es usar el Explorador de archivos y establecer la vista en "Miniaturas".

Restricciones de uso
Estos elementos comunes se usan en el desarrollo y la ilustración de imágenes personalizadas nuevas. Como parte de un lenguaje visual, estas imágenes (o cualquier parte de ellas) se deben usar de forma coherente con el nombre del archivo, que es autodescriptivo de la acción, o el verbo que la imagen representa.
